брал информацию с оф сайта МС: https://learn.microsoft.com/ru-ru/vcpkg/get_started/get-started-vs?pivots=shell-cmd

1. качаем через консоль и команду гит vcpkg, в целом в любую папку, но лучше в папку с проектом
  git clone https://github.com/microsoft/vcpkg.git

2. выполняем команду загрузки, все еще через консоль
  cd vcpkg && bootstrap-vcpkg.bat

3. добавляем команду vcpkg в PATH
  нажимаем Win+R, вводим systempropertiesadvanced и нажимаем enter
  снизу заходим в "Переменные среды" и в переменные среды пользователя добавляем путь до нашей папки vcpkg, сама папка тоже должна быть внутри пути
  нажимаем ок и все закрываем

4. качаем fmt
  в консоли vcpkg install fmt

5. редактируем CMakePresets.json
  в разделе windows-debug в cacheVariables вписываем в тулчейн путь до файла vcpkg.cmake в нашей папке
  у меня это 				"CMAKE_TOOLCHAIN_FILE": "D:/VSprojects/vcpkg/scripts/buildsystems/vcpkg.cmake"
  (47 строчка)

по сути все. это должно работать.
